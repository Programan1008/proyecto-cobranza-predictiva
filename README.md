# Sistema de Análisis Predictivo para Cobranza Bancaria

Proyecto de Ciencia de Datos end-to-end orientado a procesos de cobranza bancaria utilizando datos sintéticos y modelos de Machine Learning.

## Objetivo

Optimizar estrategias de cobranza mediante:

- Predicción de cumplimiento
- Predicción de recuperación
- Predicción de renegociación
- Segmentación inteligente de clientes

## Tecnologías

- Python 3.11
- Pandas
- NumPy
- SQLAlchemy
- SQLite
- Scikit-Learn
- XGBoost
- Power BI
- Git/GitHub

## Estructura

data/          Datos por capas
src/           Código fuente
models/        Modelos entrenados
reports/       Informes
sql/           Scripts SQL
powerbi/       Dashboard
api/           Servicio de scoring
docs/          Documentación
